# Health-InfraOps VMware VMX Templates

VMX configuration templates for manual VM creation.

## Usage

1. Copy the appropriate VMX file
2. Modify settings as needed
3. Register with ESXi host
4. Attach virtual disks

## Network Configuration

Templates include pre-configured VLAN settings:
- VLAN 10: Production
- VLAN 20: Database  
- VLAN 40: Management
